# kbot

Golang Telegram bot

# Commands

```/start hello``` - returns hello message and build version

# Installation

1. Clone this repository to Codespaces
```git clone https://github.com/alexmaidan/kbot```

2. Build code
```go build -ldflags "-X="github.com/alexmaidan/kbot/cmd.appVersion=v1.0.2```

3. Start bot
```./kbot go```
